Railway Fastener Inspection by Real-Time Machine Vision 概述

Abstract

本文讲解的是一个实时铁路紧固件检测系统，利用了一种高速激光测距相机。首先，基于像素相似性和直方图相似性的广泛的分析用在铁路轨道上。然后融合引入的不相关的方法的同时考虑性能的提升。在重复实验成功后，该方法在NI LabVIEW和实时运行上用高速3D相机在铁路运输过程中为铁路质量检测。

Introduction

1.写本文的原因

由于铁路故障隐患，人工检测有关键的缺点和隐患。1,慢；2，工作时间越长效率比较低。

该方法感兴趣的区域是六角坚固件检测。而且本文的工作是双重的。第一，自动检测六角紧固件，为铁路提供一个可靠的检测丢失螺栓的检测系统。第二，用高速3D相机去检测六角紧固件丢失问题。

系统包括：三维激光相机，激光灯光源，车轮角度编码器,收集数据必要的硬件，转换信号和处理器。

Method

本文的方法是基于像素相似性和直方图相似性。基于像素相似性的方法，为了缩小配准问题和处理噪声，输入的图像块在保证紧固件的形状的同时缩小成固定的大小然后转化成向量的形式。基于直方图相似性的方法，不能使用缩减的方法然而可以使用中值处理器做平滑操作。
基于像素相似性的方法包括主成分分析（PCA）,线性判别（LDA）,随机森林（RF）,稀疏表示(SR),多模板匹配（MTM）。

PCA:根据定义数据集的重要性将相关的样本转换成线性不相关的一系列主成分
